** 풀이 보는 방법 **

1. 온라인 풀이 URL을 브라우저에서 열기
2. 화면 우측 상단의 "Solve" 버튼 클릭
3. 자동으로 최적 풀이가 재생됩니다
4. "Step" 버튼으로 한 단계씩 확인 가능
5. "Reset" 버튼으로 초기화 후 직접 플레이 가능

화살표 표기:
- ← 왼쪽으로 이동
- → 오른쪽으로 이동  
- ↑ 위로 이동
- ↓ 아래로 이동

예: A→ = 차량 A를 오른쪽으로 1칸 이동

데이터베이스 임포트 예시 (SQL):

CREATE TABLE rush_puzzles (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    puzzle_no INTEGER,
    difficulty VARCHAR(10),
    moves INTEGER,
    board_state TEXT,
    solution_url TEXT
);

-- 파이프(|) 구분자로 임포트
COPY rush_puzzles(puzzle_no, difficulty, moves, board_state, solution_url)
FROM 'rush_puzzles_solutions.txt'
DELIMITER '|'
WHERE line NOT LIKE '=%' AND line NOT LIKE '**%' AND length(line) > 10;
